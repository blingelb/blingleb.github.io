Portfolio

Movees

The goal of this project was to take the existing design of the popular Starship delivery bots and change the mental model to create more human-to-human interaction. This project allows the user to order a Starship projector to any location on Purdue's campus and create an outdoor movie expirience. This expirience can be shared with either friends or cultivate new friendships through a shared outdoor expirience.

Value of UX Design

This exercise is intended to educate us on the values of user experience as it pertains to our aspired professional careers. To accomplish this task, we analyzed the professional field by searching through job postings in order to find common skills associated with each position. We then took to researching additional sources related to those fields for opinions on UX design written by industry-experienced professionals. From these activities we were able to compile our learnings to fully understand the importance of UX design in our future endeavors.
